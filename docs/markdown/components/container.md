# Container 容器

## 简介

带有默认内边距的容器，可将其高度设为一屏。

## API

### ContainerProps

| 属性      | 描述                                                                                                | 类型    | 默认值 |
| --------- | --------------------------------------------------------------------------------------------------- | ------- | ------ |
| full-page | 是否设置高度为屏幕高度减去头部导航高度，可用于使表格分页始终显示在屏幕底部，表格高度需设置为 100%。 | boolean | false  |

### ContainerSlots

| 插槽    | 描述           | 属性 |
| ------- | -------------- | ---- |
| default | 自定义默认内容 | -    |
